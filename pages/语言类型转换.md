- 类型转换用于将一种数据类型的变量转换为另外一种类型的变量
-
- 格式：
	- ```
	  type_name(expression)
	  ```
-
- 实例
	- ```
	  package main
	  
	  import "fmt"
	  
	  func main() {
	     var sum int = 17
	     var count int = 5
	     var mean float32
	     
	     // 将 int 转为 float32 类型
	     mean = float32(sum)/float32(count)
	     fmt.Printf("mean 的值为: %f\n",mean)
	  }
	  ```
-
- 基本类型转换
	- ```
	  字符串 -> 数字		strconv.Atoi
	  ```
